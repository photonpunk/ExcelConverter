Excel Converter

Easily convert Excel files into other formats.

Download Download View Excel Converter on GitHub

ExcelConverter creates either a tab-delimited, unicode tab-delimited, csv, or xml file from a user selected excel spreadsheet.

The unicode option parses and removes only the unwanted quotes added by the Excel export. The good quotes within a cell are retained. The Unicode option also re-encodes the exported UCS-2 16bit little endian into standard UTF-8 without BOM.
